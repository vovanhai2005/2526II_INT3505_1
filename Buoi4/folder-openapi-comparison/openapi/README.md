# OpenAPI Specification

## Prerequisites
- Python 3 installed.
- Required Python packages: `flask` and `flasgger`.

## Rendering on Localhost with Swagger UI (Flasgger)

In this project, we are using the `flasgger` Python package to serve the `openAPI.yaml` specification via Swagger UI.

1. Ensure you have installed the dependencies in your virtual environment:

```bash
pip install Flask flasgger
```

3. Run the Flask application:

```bash
python Buoi4/app.py
```

4. Open your web browser and navigate to the Swagger UI route generated by Flasgger:

```
http://127.0.0.1:5000/apidocs
```

Here, you will see your API fully documented and interactive based directly on `openAPI.yaml`.
